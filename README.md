## Recipe Share

### Frontend Setup 
```bash
# Clone repository
git clone https://github.com/tomiwa-code/recipe-share.git
cd recipe-share/frontend

# Install dependencies
```bash
npm install
```

# Start development server
```bash
npm run dev
```

# Access at http://localhost:3000
